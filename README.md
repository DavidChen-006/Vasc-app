# Vasc App

Vasc App is an early-stage mobile prototype for helping cardiovascular patients build safer, more consistent health habits through a playful, gamified experience.

The v1 concept centers on a 2D cardiovascular companion: a friendly heart-like character that acts like both a pet and a virtual twin of the patient. Instead of relying on realistic anatomy, the companion communicates status through color, expression, warmth, motion, and celebration.

## V1 Direction

The first version should focus on an individual patient experience with simulated data only.

Core v1 ideas:

- Expo, React Native, and TypeScript
- Expo Go-compatible implementation where possible
- 2D heart/cardio companion character
- Daily Cardio Score or Heart Score
- Daily missions
- XP or Pulse Points
- Heart streaks with recovery freezes or rest days
- Progress rings
- Calendar or heatmap-style progress
- Badges, milestones, and weekly recaps

## Current Safety Constraints

V1 must use simulated health data only. It should not store real patient health data, provide diagnosis, make treatment claims, integrate hardware, add AI recommendations, or introduce public leaderboards without a separate safety and privacy review.

## Development Status

The repository currently contains project documentation and agent/frontend guidance. The Expo app has not been scaffolded yet.
